# Format Author Names of a single OpenAlex work object

Formats the names of authors in a data frame.

## Usage

``` r
format_names(works, alphabetical = TRUE)
```

## Arguments

- works:

  A data frame containing author information.

- alphabetical:

  Logical indicating whether to sort the names alphabetically by last
  name (default: TRUE).

## Value

A character vector of formatted names.
